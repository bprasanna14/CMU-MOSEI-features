# AI-powered-Classic-Snake-Game
This project is under development.
Currently I have made the snake game which you play with Key strokes. I am working on the AI algorithms and soon will the functionality to it.

To play the game,

Open cmd and go to the folder where the project is clonned using cd command.
To run the game type, python snake-game.py

The play instructions are properly displayed in the game screen.
